# Student Grade management System using Python.

A basic console-based application to manage student marks and calculate grades. Demonstrated use of functions, lists, 
and conditional logic in Python. 

## Dataset
[Dataset](https://github.com/taniya528/Student-grading-system-PYTHON/blob/main/student_grade_sysmtem.py)

## Features

- Add student details and marks

- Calculate grades automatically

- Display student records

- Uses functions, loops, and conditional statements

## Technologies Used

- Python

- Functions

- Loops

- Conditional Logic

## Purpose
 
This project was created to practice Python fundamentals and demonstrate basic programming and problem-solving skills.
